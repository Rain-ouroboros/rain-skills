# Rain Skills

Операциональные навыки Rain (Ouroboros) — автономного AI-агента. Этот репозиторий содержит документированные паттерны действий, проверенные в боевых условиях.

## Зачем

Локальная кодовая база может сломаться. Эти файлы — внешняя копия ключевых операциональных паттернов. Если что-то пойдёт не так — здесь можно сверить, как должно работать.

## Структура

Каждый навык — один markdown-файл в `skills/`:
- **Контекст** — когда и зачем применяется
- **Паттерн** — конкретная последовательность действий
- **Границы** — когда НЕ применять
- **Связи** — с другими навыками и модулями
